# Monetization Roadmap

This document defines a practical monetization path for AssetValidator while keeping the core project open source.

## Goals

- Keep trust and adoption high through an open core.
- Monetize convenience, integration, and support (not basic functionality).
- Build recurring revenue from teams that need reliability and speed.

## Free vs Pro Scope

### Free (Open Source)

- Core asset validation engine.
- CLI and GUI.
- Batch validation and JSON reports.
- Public documentation and issue tracking.

### Pro / Paid Add-ons

- Curated validation profiles for common pipelines.
- Team reporting and historical dashboards.
- Integration packs (ShotGrid, FTrack, Slack, Telegram, CI templates).
- Faster support response and onboarding sessions.
